
ระบบจัดการข้อมูลและแผนที่สถานที่ฝึกงาน
(Medical Science Internship Mapping System)


1. คำอธิบายโปรเจกต์ (Project Description)
------------------------------------------------------------------
โปรเจกต์นี้คือระบบจัดการข้อมูลนิสิตฝึกงานคณะวิทยาศาสตร์การแพทย์ ที่ช่วยในการจัดเก็บ ประมวลผล และแสดงข้อมูลสถานที่ฝึกงานของนิสิตบนแผนที่แบบโต้ตอบ เพื่อให้คณะสามารถวิเคราะห์การกระจายตัวของสถานที่ฝึกงาน ประเมินความเหมาะสม และวางแผนการจัดส่งนิสิตในรอบปีถัดไปได้อย่างมีประสิทธิภาพ

2. คุณสมบัติเด่น (Key Features)
------------------------------------------------------------------
* การจัดการข้อมูลนิสิต (Student Data Management): CRUD ข้อมูลนิสิตฝึกงาน (ชื่อ-สกุล, รหัส, สาขา, ปีการศึกษา) และบันทึกข้อมูลสถานที่ฝึกงาน
* การแสดงผลบนแผนที่ (Interactive Mapping): แสดงหมุดของสถานที่ฝึกงานทั้งหมดบนแผนที่ โดยใช้ Leaflet.js คลิกที่หมุดเพื่อดูข้อมูลสรุป.
* การค้นหาและกรองข้อมูล: สามารถค้นหาและกรองข้อมูลตามสถานที่, สาขา, หรือปีการศึกษา.

3. เทคโนโลยีที่ใช้ (Technology Stack)
------------------------------------------------------------------
* Backend/Server-side: PHP (ใช้จัดการการเชื่อมต่อฐานข้อมูลและประมวลผลข้อมูล CRUD)
* Frontend/Client-side: HTML, JavaScript, CSS (ส่วนต่อประสานผู้ใช้)
* Mapping Library: Leaflet.js (สำหรับแผนที่)
* Database: MySQL/MariaDB (จัดการผ่าน phpMyAdmin)
* Web Server: Apache (ใช้ร่วมกับ XAMPP/WAMP/MAMP)

4. การติดตั้งและเริ่มต้นใช้งาน (Installation and Setup)
------------------------------------------------------------------
โปรเจกต์นี้จำเป็นต้องใช้สภาพแวดล้อมเซิร์ฟเวอร์จำลองที่มี Apache, PHP, และ MySQL (เช่น XAMPP, WAMP, หรือ MAMP)

ขั้นตอนที่ 1: การติดตั้ง Prerequisites
1. ติดตั้งโปรแกรมเซิร์ฟเวอร์จำลอง (เช่น XAMPP) และเปิดใช้งาน Apache และ MySQL
2. เข้าถึง phpMyAdmin (ปกติจะอยู่ที่ http://localhost/phpmyadmin)

ขั้นตอนที่ 2: ตั้งค่าฐานข้อมูล (Database Setup)
1. ใน phpMyAdmin ให้สร้างฐานข้อมูลใหม่ (เช่น ชื่อ internship_db)
2. Import ไฟล์ SQL (เช่น database.sql) เข้าไปในฐานข้อมูลที่สร้างขึ้น

ขั้นตอนที่ 3: คัดลอกไฟล์โปรเจกต์ (File Placement)
1. คัดลอกไฟล์ทั้งหมดจาก Repository ไปยังโฟลเดอร์หลักของ Web Server (เช่น C:\xampp\htdocs\internship-map)

ขั้นตอนที่ 4: กำหนดค่าการเชื่อมต่อ (Configuration)
1. ค้นหาไฟล์การเชื่อมต่อฐานข้อมูล (เช่น db_connect.php หรือ config.php)
2. แก้ไขค่า $servername, $username, $password, และ $dbname ให้ตรงกับการตั้งค่า MySQL ของคุณ

ขั้นตอนที่ 5: การรันโปรเจกต์ (Run the Project)
เปิด Web Browser และเข้าถึง URL ของโปรเจกต์:
http://localhost/internship-map/
(โดยที่ internship-map คือชื่อโฟลเดอร์ที่คุณคัดลอกไฟล์ไปวาง)
